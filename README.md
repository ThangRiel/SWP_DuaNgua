# Trang web đua ngựa

Đây là một giao diện web demo về nền tảng đặt cược đua ngựa với phong cách hiện đại, sang trọng và tối ưu cho trải nghiệm người dùng.

## Mục tiêu

- Trình bày các giải đua ngựa sắp diễn ra.
- Hiển thị kết quả đua ngựa gần đây.
- Cung cấp giao diện trực quan, dễ nhìn và có thể mở trực tiếp trên trình duyệt.

## Tính năng chính

- Giao diện trang chủ đẹp mắt với hiệu ứng slider nền.
- Danh sách giải đua sắp tới và tỷ lệ cược.
- Bảng kết quả cuộc đua gần đây.
- Responsive thiết kế phù hợp với máy tính và điện thoại.

## Cấu trúc thư mục

- `web/` : file giao diện chính (`index.html`).
- `styles/` : file CSS tạo kiểu cho trang.
- `imgs/` : hình ảnh nền và hình minh họa dùng trong giao diện.

## Hướng dẫn chạy

1. Mở thư mục dự án.
2. Chạy file `web/index.html` bằng trình duyệt.
   - Nếu dùng VS Code, có thể mở bằng "Go Live" hoặc extension Live Server.
3. Tùy chọn: truy cập trực tiếp bằng đường dẫn file trên máy.

## Công nghệ sử dụng

- HTML5
- CSS3
- JavaScript thuần
- Font Awesome và Google Fonts

## Ghi chú

Dự án này là phiên bản demo giao diện, chưa tích hợp backend hoặc hệ thống đặt cược thật.
